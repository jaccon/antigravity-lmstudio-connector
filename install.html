Instalação Offline

Guia passo a passo para instalar a extensão Antigravity LM Studio Connect no seu Visual Studio Code manualmente através do pacote VSIX.

1

Como gerar o arquivo VSIX (Opcional)

Se você não encontrou o arquivo .vsix na pasta (por exemplo, após baixar o código), você mesmo pode criá-lo. Basta abrir o terminal na raiz do projeto e rodar o comando abaixo para instalar a ferramenta oficial do VS Code e empacotar a extensão:

npm install -g @vscode/vsce && npm install && vsce package

Isso irá gerar um arquivo parecido com lmstudio-connector-0.0.1.vsix na pasta.

2

Acesse a aba de Extensões

Abra o Visual Studio Code. Na barra lateral esquerda, clique no ícone de Extensões ou use o atalho de teclado Ctrl+Shift+X (Windows/Linux) ou Cmd+Shift+X (macOS).

3

Selecione "Install from VSIX..."

No painel de Extensões, clique no ícone de três pontos ... (Views and More Actions) localizado no canto superior direito do menu lateral e selecione a opção Install from VSIX....

4

Navegue até o arquivo

Uma janela do explorador de arquivos será aberta. Navegue até o local onde você salvou o arquivo de extensão .vsix do Antigravity LM Studio Connect e selecione-o para instalar.

5

Instalação Concluída

Aguarde alguns segundos. O Visual Studio Code exibirá uma notificação no canto inferior direito informando que a extensão foi instalada com sucesso. Você já pode começar a utilizá-la!

Tudo pronto! Agora, basta abrir as configurações da extensão e configurar a URL ou porta local do seu LM Studio para estabelecer a conexão correta.
